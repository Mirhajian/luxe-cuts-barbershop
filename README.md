# لوکس کاتز - سایت آرایشگاه مردانه

یه پروژه ساده برای درس مهندسی نرم‌افزار دانشگاه. سیستم رزرو آنلاین نوبت آرایشگاه با React و ذخیره‌سازی لوکال.

## امکانات
- رزرو نوبت آنلاین
- مشاهده لیست خدمات و قیمت‌ها
- معرفی آرایشگران
- ذخیره داده‌ها در مرورگر (IndexedDB)

## نصب و اجرا

```bash
npm install
npm run dev
```

بعد برو به `http://localhost:5173`

## تکنولوژی‌ها
- React 18
- Vite
- Dexie.js (برای IndexedDB)

## مستندات
فایل‌های گزارش پروژه و نمودارهای UML توی پوشه `docs` هستن.
